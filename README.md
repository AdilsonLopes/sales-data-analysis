# sales-data-analysis
📊 Sales Data Analysis Project
🚀 Project Overview

In today’s data-driven world, understanding sales performance is essential for making informed business decisions.
This project focuses on analyzing sales data to uncover key trends, identify top-performing products, and detect seasonal patterns that impact revenue.

The goal is to transform raw data into actionable insights that could support strategic decision-making.

🎯 Business Problem

Companies often struggle to answer critical questions such as:

Which products generate the most revenue?
Are there regional differences in sales performance?
Is there any seasonality affecting sales?

This analysis aims to answer these questions using real-world data.

🛠️ Tools & Technologies
Python
Pandas (Data Cleaning & Analysis)
Matplotlib (Data Visualization)
Jupyter Notebook
📂 Dataset

The dataset contains sales records, including:

Order Date
Product
Region
Sales
🔍 Data Analysis Process
1. Data Cleaning
Removed duplicates
Handled missing values
Converted data types (dates and numeric fields)
2. Exploratory Data Analysis (EDA)
Total revenue calculation
Product performance analysis
Regional comparison
Time-based analysis (monthly trends)
3. Visualization
Bar charts for top products
Line charts for sales trends
Comparative analysis across regions
📊 Key Insights
The Canon imageCLASS 2200 Advanced Copier was the top revenue-generating product across most regions.
The South region showed a different behavior, where the Cisco TelePresence System EX90 led sales.
Sales showed clear seasonality, with peaks in September and November.
The business achieved a total revenue of €2.26M, indicating strong overall performance.
